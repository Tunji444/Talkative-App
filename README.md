# Talkative-App
Talkative(Talkative.py) - A basic gui python messaging app, with ports, sockets and real-time messaging



Talkative V2(Talkative_V2.py) - A basic gui ctk(custom tkinter) messaging app without real-time messaging



Basic Model(basic_model.py) - A basic terminal-based messeging .py file that has a sign up and user-messaging system
